# Windows 文件夹别名

一个基于 Go 语言和 Fyne 框架开发的图形化工具，用于在 Windows 系统上为文件夹设置自定义别名（LocalizedResourceName）。

## 功能特性

- 🖥️ **图形化界面** - 简洁易用的 GUI 操作界面
- 📁 **文件夹别名设置** - 通过修改 `desktop.ini` 文件为文件夹设置自定义显示名称
- 🔄 **资源管理器重启** - 内置资源管理器重启功能，使别名立即生效
- 🔍 **文件选择器** - 集成文件对话框，方便选择目标文件夹
- ✨ **自动配置** - 自动创建和配置 `desktop.ini` 文件（UTF-16 LE 编码）
- 🛡️ **属性设置** - 自动设置文件的系统和隐藏属性

## 技术栈

- **语言**: Go 1.26
- **GUI 框架**: Fyne v2.7.3
- **编码处理**: golang.org/x/text (UTF-16 LE 编码支持)
- **平台**: Windows

## 工作原理

本工具通过修改目标文件夹中的 `desktop.ini` 文件来设置文件夹别名。Windows 系统会读取该文件中的 `LocalizedResourceName` 属性来确定文件夹的显示名称。工具会自动：

1. 创建或编辑 `desktop.ini` 文件
2. 添加 `[.ShellClassInfo]` 节
3. 设置 `LocalizedResourceName` 属性
4. 将文件设置为系统和隐藏属性

## 安装说明

### 环境要求

- Go 1.26 或更高版本
- Windows 操作系统

### 构建步骤

1. 克隆或下载项目到本地

2. 安装依赖：
```bash
go mod download
```

3. 构建项目：
```bash
go build -o WindowsFolderAlias.exe
```

4. 准备图标文件：
确保 `icon.png` 文件存在于项目目录中

5. 运行程序：
```bash
./WindowsFolderAlias.exe
```

### 使用 Fyne 打包工具（可选）

```bash
# 安装 fyne 命令行工具
go install fyne.io/fyne/v2/cmd/fyne@latest

# 打包为可执行文件
fyne package -os windows -appType win32
```

## 使用方法

1. **启动程序** - 运行编译后的可执行文件

2. **选择文件夹** - 点击"选择文件夹"按钮，浏览并选择需要设置别名的文件夹

3. **输入别名** - 在别名输入框中输入想要的文件夹显示名称

4. **确认设置** - 点击"确认"按钮应用设置

5. **重启资源管理器**（可选） - 如果别名未立即显示，点击"重启资源管理器"按钮

## 注意事项

- ⚠️ 需要目标文件夹的写入权限
- ⚠️ 某些系统文件夹可能无法修改别名
- ⚠️ 重启资源管理器会关闭所有打开的文件资源管理器窗口
- ⚠️ 确保 `icon.png` 文件存在以正常显示程序图标
